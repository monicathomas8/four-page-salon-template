# Four Page Salon Template

A clean, reusable 4 page static website template built with HTML, CSS and JavaScript. 
Designed for salon and wellness businesses but easily adaptable for any small business.

## Pages
- Home
- About
- Services
- Contact
- Privacy & Cookies (footer)

## Built With
- HTML
- CSS
- JavaScript

## Demo Project
This template was built using a fictional salon called **Cuts by Janice** as the demo project.

## How To Use This Template
1. Click **Use this template** on GitHub
2. Create your new repo
3. Clone it down
4. Customise the content and CSS for your client
5. Deploy

## Deployment
This template can be deployed on any static hosting platform. 
Netlify is currently recommended as it handles the contact form out of the box with Netlify Forms.
